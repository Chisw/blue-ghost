## Blue Ghost

一个小巧的 Chrome 扩展

### 书签拼音首字母快搜

按拼音首字母、全拼快速搜索

### 魔页

### 自定义浏览器 New Tab

指定新开标签页页面，与浏览器自带设置的不同之处，在于可以隐藏 URL
